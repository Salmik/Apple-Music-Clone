# Apple-Music-Clone

**Simple apple music app clone**

## Installation

```
run - pod install
```

## Technologies
SDWebImage - [SDWebImage](https://github.com/SDWebImage/SDWebImage)<br/>
Alamofire - [Alamofire](https://github.com/Alamofire/Alamofire)<br/>
